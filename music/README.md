# Music folder (optional, recommended)

Drop 5–15 background tracks here (`.mp3`/`.wav`/`.m4a`/`.ogg`) — each run picks one at
random, loops it, and ducks it under the narration.

**Where to get monetization-safe tracks ($0):**

1. Go to **studio.youtube.com → Audio Library** (needs your channel's Google login).
2. Filter Genre: *Ambient / Cinematic*, Mood: *Calm / Inspirational*.
3. Check the **License type** column — prefer tracks that don't require attribution.
   If you use a **CC-BY** track, paste the artist credit into your video description
   (the run's `metadata.md` reminds you).
4. Download → upload the files to this folder via GitHub's web UI (**Add file → Upload files**).

No files here? Fine — videos render without music. Set `music.volume: 0` in
`config.yaml` to disable music explicitly.

> Why not automate music download too? YouTube's Audio Library has no public API, and its
> terms require downloads through the official page. One manual batch of ~10 tracks lasts
> months of runs.
